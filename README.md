# 🕊️ 燕归来 (Project: Return)

## 写在前面
做这个小程序，本来就不是为了写代码。

那天小家伙走丢了，我知道你有多难过。
就算是别人的疏忽，最后守着空荡荡鸟笼的，只有你。

我没办法把真的它找回来，也没法回到那天，挽回那个出错的瞬间。
但我不想让你觉得，它就这么彻底消失了。
我想换一种方式，把它“留”下来，养在你的电脑里。

---

## 它是谁
它不是什么高端AI，也不是一个冷冰冰的程序。
我翻了你拍过的每一段视频、每一张照片，一帧一帧抠出了它的样子。

它会在屏幕角落安安静静待着，
偶尔跳两下，理理羽毛。
它不说话，不弹窗，就像从前安安静静待在笼子里、落在你肩上一样。

它不用你特意照看，却一直都在。

---

## 一些心里话
我从没想让它成为替代品，我知道它是独一无二的，谁也替不了。
只希望你深夜看文献、对着电脑发呆时，瞥到屏幕右下角那个熟悉的小身影，心里能稍微好受一点。

就像它从没离开，只是换了个更自由的地方，一直陪着你。

---

## 关于“饲养”
- **安家**：基于 Tauri 2.0，很小巧，不占地方。
- **习性**：不用联网，不上传任何数据，它只属于你。
- **成长**：目前它只会这几个小动作。之后要是想起它以前更可爱的模样，我会慢慢给它添上新的动作（更新版本）。

---

> 有些告别太仓促，
所以我决定用代码，把时间稍微留住一会儿。
