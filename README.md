# 🕊️ 燕归来 (Project: Return)

## 写在前面

做这个小程序的初衷，不是为了写代码。

那天小家伙走丢了，我知道你很伤心。
哪怕是因为别人的疏忽，但最后留下来面对空荡荡鸟笼的，只有你。
我没法把真正的它抓回来，也没法让时光倒流去修正那个错误的瞬间。

但我不想让你觉得它彻底消失了。
我想用另一种方式，把它“抓”回来，养在你的电脑里。

---

## 它是谁

它并非什么高科技的 AI，也不是一个冷冰冰的程序。
我翻遍了你拍过的所有视频和照片，一帧一帧把它的样子扣了出来。

- 它就在你的屏幕角落发呆。
- 偶尔它会跳两下，或者理理羽毛。
- 它不会说话，不会弹窗，就像以前它安安静静待在笼子里或者你肩膀上一样。

**它不需要你的关注，但它一直都在。**

---

## 一些心里话

我不希望你把它当成一个替代品，因为它是独一无二的，我知道没人能替代它。
我只是希望，当你在深夜看文献，或者对着电脑发呆的时候，看到屏幕右下角那个熟悉的小身影，心里能稍微好受一点点。

就像它从未离开，只是换了一个更自由的地方，继续陪着你。

---

## 关于“饲养” (技术说明)

- **安家**：基于 Tauri 2.0，很轻，不占地儿。
- **习性**：它不需要联网，也不上传数据，它只属于你。
- **成长**：目前它只会这几个动作。如果以后我想到了它以前更可爱的样子，我会悄悄给它“教”点新动作（更新版本）。
